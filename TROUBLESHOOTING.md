# Troubleshooting Guide

## Common Issues

### API Key Not Working
- Verify your OpenAI API key is correct
- Check that you have sufficient credits
- Ensure the key hasn't expired

### Stock Symbol Not Found
- Verify the symbol is correct (e.g., AAPL not APPL)
- Check that the stock is publicly traded
- Try a different symbol format

### App Not Loading
- Ensure all dependencies are installed: `pip install -r requirements.txt`
- Check Python version (3.8+ required)
- Try clearing browser cache

### Analysis Taking Too Long
- Check your internet connection
- Verify YFinance API is accessible
- Try with different stock symbols

## Getting Help

If issues persist, please open an issue on GitHub with:
- Error messages
- Steps to reproduce
- Your environment details
