A powerful Streamlit-based investment analysis tool that uses AI to compare stocks and generate detailed investment reports.

## Features

- **Stock Comparison**: Compare two stocks side-by-side
- **AI-Powered Analysis**: Uses GPT-4o to analyze stock performance
- **Comprehensive Reports**: Get detailed reports on stock prices, analyst recommendations, and fundamentals
- **User-Friendly Interface**: Clean and intuitive Streamlit UI

## Installation

1. Clone the repository:
```bash
git clone https://github.com/Shriiii01/investment_agent.git
cd investment_agent
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the application:
```bash
streamlit run agent.py
```

## Usage

1. Enter your OpenAI API key in the app
2. Enter two stock symbols (e.g., AAPL and MSFT)
3. Click to analyze and get a detailed comparison report

## Requirements

- Python 3.8+
- OpenAI API key
- Internet connection for fetching stock data
