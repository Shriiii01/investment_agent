# Quick Start Guide

Get up and running with the Investment Agent in 3 simple steps!

## Step 1: Install Dependencies

```bash
pip install -r requirements.txt
```

## Step 2: Get Your API Key

You'll need an OpenAI API key. Get one at https://platform.openai.com/api-keys

## Step 3: Run the App

```bash
streamlit run agent.py
```

## Example Usage

1. Enter your OpenAI API key
2. Type two stock symbols (e.g., `AAPL` and `MSFT`)
3. Click "Analyze Stocks"
4. Review your detailed comparison report
